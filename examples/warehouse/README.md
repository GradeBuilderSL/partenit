## Warehouse Example

This directory will contain:
- example safety policies for a warehouse robot
- simulation scenarios for `partenit-safety-bench`
- scripts or notebooks demonstrating guard behavior in typical warehouse workflows

It will be a primary reference for new users exploring Partenit in a realistic but self-contained domain.
